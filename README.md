# ∇Dash - Trading Journal & Analytics  
![Screenshot](assets/screenshot.png)  

## Features  
- Trade logging (SQLite backend)  
- Real-time trading analytics  
- Image uploads for trade receipts  

## Run Locally  
```bash
pip install -r requirements.txt  
python app.py
```
